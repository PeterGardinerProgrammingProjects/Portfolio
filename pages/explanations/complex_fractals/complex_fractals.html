Complex Fractals

You will need to understand complex numbers to understand this explanation. Fractals are mathematical objects which have detail at any scale. Usually, if you zoom into a section of a graph or a surface, it approaches a line or a plane. Often, fractals are self-similar, meaning that it contains smaller copies of itself, but this is not a requirement.

The Mandelbrot Set is a fractal in the complex plane. It is likely the most famous fractal, and it is defined, in part, by this equation.

$$ z_n=z_{n-1}^2+c $$

The Mandelbrot Set is a set of complex numbers. This equation is used to determine whether or not a particular number, c, is in the set. It starts with z = 0. Through this recursive function, we can find the next value of z, and the next, and so on. Once we have evaluated the function an infinite number of times, we cand decide whether or not c is in the set. If z grew without bound (reached infinity), then c is not in the Mandelbrot Set. Otherwise, it is. Let's see an example with c = 1 + 0i. Remember, z starts at zero.

$$ z_1=z_{0}^2+1=1 $$

$$ z_2=1^2+1=2 $$

$$ z_3=2^2+1=5 $$

$$ z_4=5^2+1=26 $$

It is clear that c = 1 + 0i is not in the Mandelbrot set, because this value causes z to grow endlessly. There are more complex fractals, like Julia sets, which are related to the Mandelbrot Set.
